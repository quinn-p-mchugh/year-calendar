# YearSpread: Visualize your year in a spreadsheet

**YearSpread** is a yearly calendar template to help you plan out your year(s) in a digital format. Use it to block out time for all-day events, weekend trips, vacations, etc. and/or keeping track of annual events like festivals or conferences.
